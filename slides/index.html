Solution · one sign-in

One sign-in.
Every brand.

One Microsoft Entra session — and the cockpit picks up where Sara left off. Trusted store device. No password copy. Sessions designed to outlive a fitting-room queue.

  • SSO only — no password UI ever shown
  • Welcome back — last shift closed time auto-restored
  • Role picker — four roles wired, each with its own brief
  • Session live — pulsing indicator the floor can trust

Solution · live pulse
